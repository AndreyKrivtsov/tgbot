Телеграм бот, подключаемый в группы, который:

* Показывает капчу (задание), которое просит пользователя выбрать правильное решение задачи. При выполнении задания, пользователь остается в группе, в других случаях пользователь кикается из группы
* Модерирует чат с помощью AI-агента
* Имеет веб-интерфейс в виде TG WebApp для настройки

Процесс добавления в группу:

* Модератор группы пишет в чат команду /register
* Бот добавляет группу в базу активных и запоминает текущих модераторов группы
* После добавления сразу работает Капча
* Для включения AI агента

Процесс удаления из группы:

* Модератор группы пишет команду /unregister
* Бот очищает данные группы из памяти
